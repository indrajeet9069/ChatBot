
# Chat Bot

A ChatBot capable of learning things on its own, opening applications, playing music & our daily tasks.


## Features

- Playing Music.
- Can set reminder.
- Change wallpaper.
- Changing its own Name.
- Opening Applications/Websites.
- Checking internet connectivity.
- And a lot more!
